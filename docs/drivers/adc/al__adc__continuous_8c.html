◆ AlAdc_ContinuousMode_Test()

static AL_S32 AlAdc_ContinuousMode_Test ( AL_VOID  )
static

This function initializes the ADC hardware with specified configurations, starts the ADC in continuous mode, and reads data from all configured channels for a number of iterations. It logs the value of each channel for each iteration. After completing the iterations, it stops the ADC and cleans up.

Parameters
None.
Returns
AL_S32 Returns AL_OK if the ADC continuous mode operates correctly, or an error code if initialization or starting the ADC fails.

al_adc_continuous.c.
