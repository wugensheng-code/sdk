ADC API Documentation

The AL9000 ADC driver. This driver supports the AL9000 ADC Controller.

The ADC Controller supports the following features:

  • ADC_PL
    • 支持多种adc_clk来源（osc/PS/fabric），可切换，同时支持分频模式（adc_clk的2-256分频）
    • 支持低功耗模式，adc时钟可以gate
    • 支持多个模拟输入来源，包括16个复用IO和1个专用IO
    • 支持来自config和ADC_PS的APB信号的切换
    • 支持APB和APB的仲裁，优先JTAG转换的APB访问
    • 支持single channel mode，可以指定channel
    • 支持 single pass mode，扫描一遍所有使用的channel数据
    • 支持continuous mode，持续读取所有使用的channel数据
    • 支持 external mux mode ，会输出一组3bit的寄存器数据，用于指定外部的channel地址
    • 支持大于或小于指定阈值的中断，每个channel可独立设置
    • 支持可以mask的中断
  • ADC_PS
    • JTAG时钟TCK支持多种分频模式（apb_clk的2/4/8/16分频）
    • 待发送的command信号支持 16深度的FIFO
    • 接收的data数据支持16深度的FIFO
    • 支持串并，并串转换
    • 支持FIFO空满的中断
    • 支持可以mask的中断
