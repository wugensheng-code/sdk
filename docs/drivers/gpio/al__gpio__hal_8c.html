◆ AlGpio_Hal_EnablePinDebounce()

static AL_S32 AlGpio_Hal_EnablePinDebounce ( AL_GPIO_HalStruct *  Handle,
AL_U32  Pin 
)
static

This function enables pin debounce for a specific GPIO pin. Pin debounce is a technique used to eliminate the effects of mechanical switch bouncing, which can cause multiple false readings when a switch is pressed or released. By enabling pin debounce, the GPIO pin will only register a single stable state change, ignoring any temporary fluctuations caused by switch bouncing.

Parameters
HandlePointer to the AL_GPIO_HalStruct representing the GPIO handle.
PinThe GPIO pin number to enable debounce for.
Returns
Returns AL_OK if the pin debounce was successfully enabled, or an error code if an error occurred.

◆ AlGpio_Hal_SetPinIntrType()

static AL_S32 AlGpio_Hal_SetPinIntrType ( AL_GPIO_HalStruct *  Handle,
AL_U32  Pin,
AL_GPIO_IntrEnum  IntrType 
)
static

This function sets the interrupt type for a specific GPIO pin. The interrupt type can be one of the following:

  • GPIO_INTR_TYPE_EDGE_RISING: Interrupt triggered on rising edge.
  • GPIO_INTR_TYPE_EDGE_FALLING: Interrupt triggered on falling edge.
  • GPIO_INTR_TYPE_EDGE_BOTH: Interrupt triggered on both rising and falling edges.
  • GPIO_INTR_TYPE_LEVEL_HIGH: Interrupt triggered when the pin is at a high level.
  • GPIO_INTR_TYPE_LEVEL_LOW: Interrupt triggered when the pin is at a low level.
Parameters
HandlePointer to the AL_GPIO_HalStruct instance.
PinThe GPIO pin number.
IntrTypeThe interrupt type to set.
Returns
AL_OK if successful, an error code otherwise.
