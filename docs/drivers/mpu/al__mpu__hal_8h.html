AL_U8  ConfigNumber

This function initializes the MPU device with a specified configuration. It looks up the hardware configuration based on the device ID, registers an event callback, and initializes the device with the provided configuration. It returns an error code if any step fails.

Parameters
MpuDevIdThe device ID of the MPU.
HandlePointer to a pointer to the MPU HAL structure to be initialized.
EventCallBackThe event callback function to be registered.
InitRegionConfigPointer to the initial region configuration structure.
ConfigNumberThe number of configurations to apply.
Returns
AL_OK on success, an error code otherwise.

al_mpu_example.c.
