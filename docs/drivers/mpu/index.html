AL9000 Project
MPU API Documentation

The AL9000 MPU driver. This driver supports the AL9000 MPU Controller.

The MPU Controller supports the following features:

  • 支持基于地址段的保护，保护段 数参数化配置，范围1~32
  • 每个地址段支持1bit使能控制，仅使能时保护开启
  • 支持配置根据读写维度保护，包含： 1）读保护 写保护 2） 只读保护 3） 只写保护 4） 读写均不保护
  • 支持根据命令安全属性保护，包含： 1） 仅安全访问 2）安全非安全均可访问
  • 支持基于特权模式的保护，包含： 1） 仅privileged访问 2）privileged和normal均可访问
  • 支持根据groupid维度的保护，groupid用于区分针对不同master的保护行为
  • 支持中断，中断区分不同MPU以及不同region
