static AL_S32 AlTtc_Test_PwmOutput ( AL_VOID  )
static

This function initializes the TTC device with specific configurations for PWM output, sets up the interval and match values to generate PWM signal, and enables the waveform output. It configures the device to generate a PWM signal with a specific frequency and duty cycle, based on predefined values for the interval max value and match value.

Returns
AL_OK on successful initialization and configuration, or an error code on failure.
