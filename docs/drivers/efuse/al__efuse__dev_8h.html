AL_VOID AlEfuse_Dev_SetBits ( AL_U32  Offset,
AL_U32  Num,
AL_U32 *  Val 
)

This function sets bits in the efuse device starting from a given offset. If the Val parameter is NULL, it sets the specified number of bits to 1. Otherwise, it sets the bits according to the values in the Val array. The function also logs the operations performed.

Parameters
OffsetThe starting offset where bits will be set.
NumThe number of bits to set.
ValA pointer to an array of values to set the bits. If NULL, all bits are set to 1.
Returns
AL_VOID This function does not return a value.
