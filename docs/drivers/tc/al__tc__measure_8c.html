static AL_S32 AlTc_Test_MeasurePulseWidth ( AL_VOID  )
static

This function initializes the TC hardware with predefined configurations, sets up the event timer for pulse width measurement, and continuously reads and logs the event timer value. The function runs indefinitely, logging the value of the event timer.

Returns
AL_OK if the TC hardware is successfully initialized, error code otherwise.
