TC API Documentation

The Xilinx TC driver. This driver supports the Xilinx TC Controller.

The TC Controller supports the following features:

  • 支持3个独立可编程的16位定时器/计数器
  • 支持16位预分频器，用于时钟速率选择
  • 支持三个中断，每个计时器计数器各一个
  • 支持在溢出或定期间隔时产生的中断
  • 支持当计数匹配可编程值时产生中断
  • 支持一个事件计时器测量每个外部时钟脉冲的长度(高或低持续时间)
  • 支持计数器递增或递减
  • 支持随时读取每个计数寄存器的当前值
  • 支持通过MIO和PL产生波形输出
  • 支持地址保护
