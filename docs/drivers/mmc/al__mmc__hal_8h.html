AL_S32 AlMmc_Hal_Init ( AL_MMC_HalStruct **  Handle,
AL_U32  DevId,
AL_MMC_InitStruct *  InitConfig,
AL_MMC_EventCallBack  CallBack 
)

This function configures the MMC device with the specified initialization parameters, registers a callback for MMC events, and sets up interrupt handling for the device. It looks up the hardware configuration based on the device ID, initializes the MMC device, and registers the event callback. If no callback is provided, a default callback is registered. Finally, it registers the interrupt handler for the device.

Parameters
HandleDouble pointer to the MMC HAL structure, which will be initialized by this function.
DevIdDevice ID for the MMC device. Used to look up hardware configuration.
InitConfigPointer to the initialization configuration structure.
CallBackEvent callback function to be registered.
Returns
Returns AL_OK on success, AL_MMC_ERR_NULL_PTR if the handle is NULL, AL_MMC_ERR_INVALID_DEVID if the device ID is invalid, or other error codes as defined.
Examples:
al_mmc_adma2_blocked.c, al_mmc_non_dma_blocked.c, and al_mmc_sdma_blocked.c.

This function reads a specified number of blocks from the MMC device starting at a specified block offset. It waits until the read operation is complete or a timeout occurs.

Parameters
HandlePointer to the MMC HAL structure.
BufPointer to the buffer where the read data will be stored.
BlkOffsetBlock offset at which to start reading.
BlkCntNumber of blocks to read.
TimeoutTimeout for the read operation in milliseconds. Currently unused.
Returns
Returns AL_OK on success or an error code on failure.
Examples:
al_mmc_adma2_blocked.c, al_mmc_non_dma_blocked.c, and al_mmc_sdma_blocked.c.
