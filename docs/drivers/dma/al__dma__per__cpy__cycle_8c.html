static AL_S32 AlDma_Test_PerCpyCycle ( AL_VOID  )
static

This function initializes DMA channels for memory-to-peripheral and peripheral-to-memory transfers, sets up local interrupts, performs data copying from source to destination using DMA, checks the data integrity, and cleans up resources.

Returns
AL_OK if the test and data verification pass, otherwise returns an error code.
