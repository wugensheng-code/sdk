UART API Documentation

The AL9000 UART driver. This driver supports the AL9000 UART Controller.

The UART Controller supports the following features:

  • 支持可编程的波特率发生器
  • 支持64字节接收和发送FIFO
  • 支持可编程的协议：数据位数（6，7，8）；可选的奇偶校验位；停止位数（1，1.5，2）
  • 支持奇偶校验，帧和溢出错误检测
  • 支持线终止的产生与检测
  • 支持中断的产生
  • 支持环回模式
  • 支持两种可编程模式的DMA
  • 支持自动流控制
  • 调制解调器控制信号：CTS、RTS、DSR、DTR、RI和DCD仅在EMIO接口上可用
