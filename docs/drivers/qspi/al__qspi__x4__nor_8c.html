◆ AlNor_SetQuad1ByteMode()

AL_S32 AlNor_SetQuad1ByteMode ( AL_U8  SetQuadCmd,
AL_U8  ReadQuadCmd,
AL_U8  QuadPos 
)

Enables the Quad mode in the NOR flash memory by setting a specific bit in a configuration register. This function first reads the current value of the configuration register, modifies it to enable Quad mode, and then writes it back to the NOR flash. It ensures the NOR flash is ready for write operations before and after modifying the configuration register by calling AlNor_Wren() and AlNor_WaitWip().

Parameters
SetQuadCmdThe command to write the configuration register to enable Quad mode.
ReadQuadCmdThe command to read the current value of the configuration register.
QuadPosThe bit position in the configuration register that enables Quad mode.
Returns
AL_OK if the operation was successful, otherwise an error code.
