This function initializes an interrupt for inter-process communication by setting up the handle, type, callback function, and argument.

Parameters
HandleDouble pointer to the AL_IpcIntr_HalStruct structure where the initialized interrupt will be stored.
IntrTypeThe type of the interrupt to initialize.
FuncPointer to the callback function to be called when the interrupt occurs.
ArgPointer to the argument to be passed to the callback function.
Returns
AL_OK on success, AL_IPC_ERR_ILLEGAL_PARAM if the parameters are invalid or the hardware configuration is not found.
